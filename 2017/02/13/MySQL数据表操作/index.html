打开数据库

USE 数据库名称;

显示当前用户打开的数据库

SELECT DATABASE();

创建数据表

CREATE TABLE [IF NOT EXISTS] table_name (column_name data_type,…)
举例:

查看数据库中的数据表

SHOW TABLES [FROM db_name] [LIKE ‘pattern’ | WHERE expr];

查看数据表结构

SHOW COLUMNS FROM tb_name;

记录的插入

INSERT [INTO] tb_name [(col_name,…)] VALUES(val,…);

举例:

INSERT INTO tb VALUES(‘TOM’,25,7843.25);

INSERT INTO tb (username,salary) VALUES(‘John’,4500.33);

记录的查询

SELECT * FROM tb;

MySQL空值(NULL)与非空值(NOT NULL)

NULL, 字段值可以为空.

NOT NULL, 字段值禁止为空.

举例:

CREATE TABLE IF NOT EXISTS tb (username VARCHAR(20) NOT NULL, age TINYINT);

INSERT INTO tb (NULL,20);#会报错,username不能为空.

MySQL的自动编号(AUTO_INCREMENT)

自动编号, 且必须与主键组合使用.

默认情况下, 起始值为1, 每次的增量为1.

MySQL主键约束(PRIMARY KEY)

1.一张数据表中只能存在一个主键.

2.主键保证记录的唯一性.

3.主键自动为NOT NULL.

MySQL唯一约束(UNIQUE KEY)

1.唯一约束可以保证记录的唯一性.

2.唯一约束的字段可以为空值(NULL).

3.每张数据表可以存在多个唯一约束.

MySQL默认约束(DEFAULT)

1.当插入记录时,如果灭有明确为字段赋值,则自动赋予默认值.
