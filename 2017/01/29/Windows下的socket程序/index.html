源码解析

WinSock(Windows Socket)编程依赖于系统提供的动态链接库(DLL),有两个版本:

1)较早的DLL是wsock32.dll, 大小为28kb, 对应的头文件为winsock1.h

2)最新的DLL是ws2_32.dll, 大小为69KB, 对应的头文件为winsock2.h.

几乎所有的Windows操作系统都已经支持ws2_32.dll. 所以我们都可以使用ws2_32.dll

使用DLL之前必须把DLL加载到当前程序, 你可以在编译时加载, 也可以在程序运行时加载.

#pragma comment (lib, “ws2_32.lib”) //这里是编译时加载.

3) 调用WSAStartup()函数进行初始化, 以指明WinSock规范的版本.

int WSAStartup(WORD wVersionRequested, LPWSADATA lpWSAData);
