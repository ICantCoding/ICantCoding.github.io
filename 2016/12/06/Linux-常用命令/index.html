Linux 文件目录操作常用命令

Linux 常用命令！那是必须掌握的技能呀！不多说！一起来看看吧！



  • 1. ls -ladih /etc
    选项 d 用于显示目录本身的信息,而不显示目录下文件的信息！
    选项 i 用于查看文件的inode节点, 在linux下, 系统是根据inode节点(有点想文件的索引号)来查找文件的！
    选项 h 表示文件显示信息人性化, 结合选项l使用可以查看文件的具体大小！



  • 2. mkdir -p /tmp/mulu0
    选项 p 用于递归创建目录, 即在不存在的目录下, 创建子目录.

    mkdir /tmp/mulu1 /tmp/mulu2
    支持同时创建多个目录.


  • 3. pwd
    显示当前工作目录的绝对路径, pwd全拼 print working directory.


  • 4. rmdir /tmp/mulu0

    删除空目录, 注意, 用于删除空目录！不能删除非空的目录！


  • 5. cp -rp [原目录/文件] [目标目录]

    选项 r 用于复制目录

    选项 p 用户保持文件的属性

    cp /root/install.log /root/install.log.syslog /tmp

    可以复制多个文件到一个目录下

    cp -r /tmp/mulu1 /root/mulu11

    复制文件目录的同时支持改名!


  • 6. mv [原目录/文件] [目标目录]

    剪切和改名

    mv /tmp/mulu1 /root
    剪切

    mv /tmp/mulu1 /tmp/mulu11
    改名



  • 7. rm -rf [文件或目录]

    选项 -r 删除目录

    选项 -f 强制执行删除, 不会问你要不要确定删除！！



  • 8. touch filename.txt

    创建一个文件

    touch program files

    创建了两个文件, 分别是 program 和 files

    touch “program files”

    创建了一个文件, 叫 program files


  • 9. cat -n /etc/issue

    查看文件内容, 选项 -n 显示行号!


  • 10. tac /etc/issue

    倒着查看文件内容.


  • 11. more /etc/issue
    分页显示文件内容, 进入more页面后 空格或f翻页, Enter换行, q或Q退出, 奇葩之后向后翻页


  • 12. less /etc/issue
    跟more差不多, 而且支持向上翻页, 也支持搜索, /xxx


  • 13. head -n 7 /etc/services

    显示文件的前7行


  • 14. tail -n 3 /etc/services

    查看文件的最后3行

    tail -f /var/log/messages

    动态显示文件末尾内容！动态实时变化！


  • 15. ln -s [原文件] [目标文件]

    生成链接文件

    ln -s /etc/issue /tmp/issue.soft

    创建文件/etc/issue的软链接/tmp/issue.soft

    ln /etc/issue /tmp/issue.hard

    创建文件/etc/issue的硬链接/tmp/issue.hard, 感觉跟cp -p一样的效果, 但是必须指出,硬连接的这两个文件会同步更新！硬链接之所以能同步更新, 是因为硬链接的这两个文件具有相同的inode节点。对其中一个文件的修改， 系统会根据inode节点来映射到两个文件.注意硬链接不能跨分区，而软链接可以跨分区, 硬链接不能针对目录使用，而软链接可以。
