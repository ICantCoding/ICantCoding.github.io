Linux网络命令

网络命令, 有点多, 但在互联网时代, 网络是多么重要, 必须学好它！

write

write <用户名>

给用户发送信息, 以ctrl+D保存结束. 注意, 这里的用户,必须是要登录同一个服务器的用户, 我们可以用who或w命令查看, 该linux服务器当前有哪些用户登录在线.

该指令的路径: /usr/bin/write

wall

wall [message]

相当于write all命令, 给所有人发送信息. 自己也会收到该信息.

ping

ping [-c] ip地址

给远程主机发送数据包, 测试网络连通性.

选项-c 指定发送多少次包

该指令的路径: bin/ping

ifconfig

ifconfig 网卡名称 IP地址

查看和设置网卡信息, 全称:interface configure, 注意使用ifconfig来修改IP地址是临时生效的.

该指令所在路径: /sbin/ifconfig

mail

mail [用户名]

查看发送电子邮件!

mail查看邮件, help查看mail帮助, h返回邮件列表, 数字来查看第几封邮件.

last

last

列出目前与过去登录系统的用户信息

该命令的路径: /usr/bin/last

lastlog

lastlog

先把所有的用户列出来, 并显示他们最后一次登录系统的信息

last -u 502

502是用户的uid, 是用户在linux系统中的标识, 该命令查看502用户最后一次登录系统的信息.

traceroute

traceroute [访问地址]

显示数据包到主机间的路径.

该命令的路径: /bin/traceroute

netstat

netstat

显示网络相关信息

该命令路径: /bin/netstat

选项-t : TCP协议

选项-u : UDP协议

选项-l : 监听

选项-r : 路由

选项-n : 显示IP地址和端口号

范例:

netstat -tlun 查看本机监听的端口

netstat -an 查看本机所有的网络连接

netstat -rn 查看本机路由表

挂载命令 mount

mount [-t 文件系统] 设备文件名 挂载点

mkdir /mnt/cdrom 先创建挂载点

mount -t iso9660 /dev/sr0 /mnt/cdrom/ 再挂载, 注意要先写设备文件名

mount /dev/sr0 /mnt/cdrom/ 其实可以将-t iso9660删除掉

cd /mnt/cdrom/ 去挂载点下使用就可以了

umount /mnt/cdrom/ 取消挂载, 注意umount后边的参数可以是挂载点位置, 也可以是设备位置, 但是要注意不能在挂载点的目录下来取消挂载, 必须退出挂载点目录来取消挂载.
